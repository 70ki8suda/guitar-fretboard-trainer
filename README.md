# Guitar Fretboard Trainer

Guitar Fretboard Trainer is a single-screen practice app for learning scale tones on a standard 6-string guitar fretboard. It shows scale degrees and movable-do solfege across frets 0 through 21, with color coding that makes interval patterns easy to scan.

## Setup

```bash
pnpm install
```

## Commands

```bash
vp dev
vp check
vp test
vp build
```

## Notes

The UI was shaped by StyleX patterns and Nodex references used during implementation, especially for layout, token structure, and visual tone.
